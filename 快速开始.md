# 快速开始指南

## 5 分钟上手

### 第 1 步：打开应用（10 秒）

双击 `index.html` 文件，在浏览器中打开。

### 第 2 步：添加第一个企业（30 秒）

1. 点击"待研究企业"下的"+ 添加"按钮
2. 填写企业名称（如：阿里巴巴）
3. 点击"保存"

恭喜！你已经添加了第一条记录。

### 第 3 步：尝试快速录入（1 分钟）

1. 打开招聘网站（如 Boss 直聘、拉勾网）
2. 复制一条招聘信息，例如：
   ```
   字节跳动 - 前端开发工程师
   地点：北京
   薪资：30-50K
   ```
3. 点击顶部"⚡ 快速录入"按钮
4. 粘贴刚才复制的内容
5. 点击"🔍 本地识别"
6. 确认识别结果后点击"保存"

### 第 4 步：移动职位（30 秒）

1. 点击卡片上的"➡️"箭头按钮
2. 或者直接拖拽卡片到其他池子

### 第 5 步：创建简历（1 分钟）

1. 点击顶部"📄 简历管理"
2. 点击"+ 新建简历"
3. 填写简历名称（如：互联网通用版）
4. 点击"保存"

### 第 6 步：记录投递（1 分钟）

1. 将职位移动到"已投递企业"池
2. 点击卡片上的"✏️"编辑按钮
3. 选择使用的简历
4. 填写投递日期
5. 设置投递状态
6. 点击"保存"

### 第 7 步：查看进展（30 秒）

点击顶部"📈 投递进展"，查看统计数据和时间线。

## 进阶使用（可选）

### 配置 AI 智能识别

如果你想要更准确的信息识别：

#### 方案一：DeepSeek（推荐，国内可用）

1. 访问 https://platform.deepseek.com/
2. 注册账号（手机号即可）
3. 创建 API Key
4. 在应用中点击"⚙️ 设置"
5. 选择"DeepSeek"
6. 粘贴 API Key
7. 点击"保存配置"
8. 点击"测试连接"

**费用**：约 0.002 元/次，新用户赠送 500 万 tokens

#### 方案二：本地模型（免费）

1. 安装 Ollama：https://ollama.ai/download
2. 下载模型：`ollama pull llama2`
3. 启动服务：`ollama serve`
4. 在应用设置中选择"本地模型"
5. 保存配置

**优点**：完全免费，数据不出本地

## 常见使用场景

### 场景 1：批量收集岗位信息

**目标**：快速收集 50 个感兴趣的岗位

**步骤**：
1. 打开招聘网站
2. 浏览岗位，看到感兴趣的就复制信息
3. 使用"快速录入"功能批量添加
4. 全部添加到"待研究企业"池
5. 周末再慢慢筛选

**时间**：约 30 分钟

---

### 场景 2：精准投递

**目标**：针对不同类型的公司使用不同简历

**步骤**：
1. 准备 3 个简历版本：
   - 互联网大厂版
   - 传统企业版
   - 创业公司版
2. 在"简历管理"中录入这 3 个版本
3. 投递时选择对应的简历版本
4. 记录投递日期和状态

**效果**：提高简历通过率

---

### 场景 3：面试进度追踪

**目标**：清晰了解每个投递的最新状态

**步骤**：
1. 收到查看通知 → 更新状态为"已查看"
2. 收到面试邀请 → 更新为"面试邀请"，备注中记录时间
3. 面试完成 → 更新为"已面试"，备注中记录反馈
4. 收到 offer → 更新为"已通过"
5. 收到拒信 → 更新为"已拒绝"

**效果**：不会错过任何机会

---

### 场景 4：数据分析优化

**目标**：分析投递效果，优化策略

**步骤**：
1. 每周查看"投递进展"
2. 计算查看率、面试率、通过率
3. 对比不同简历版本的效果
4. 分析哪些类型的公司回复率高
5. 调整投递策略

**指标参考**：
- 查看率 > 50%：简历质量不错
- 面试率 > 20%：岗位匹配度好
- 通过率 > 5%：面试表现良好

## 故障排除

### 问题 1：页面打不开

**可能原因**：
- 浏览器不支持
- 文件路径有中文或特殊字符

**解决方法**：
- 使用 Chrome、Edge、Firefox 等现代浏览器
- 将文件夹移到英文路径下

### 问题 2：数据不见了

**可能原因**：
- 清除了浏览器数据
- 换了浏览器
- 使用了隐私模式

**解决方法**：
- 从备份文件导入
- 使用同一个浏览器
- 不要使用隐私/无痕模式

### 问题 3：AI 识别失败

**可能原因**：
- API Key 未配置或错误
- 网络连接问题
- API 额度用完

**解决方法**：
- 检查 API Key 配置
- 点击"测试连接"
- 使用"本地识别"作为备用

### 问题 4：识别结果不准确

**可能原因**：
- 输入文本格式不规范
- 本地识别能力有限

**解决方法**：
- 使用 AI 识别
- 手动修改识别结果
- 直接使用"+ 添加"功能
