## 升级指南

### 从 v1.0 升级到 v2.0

#### 方法一：自动迁移（推荐）

1. 下载新版本文件
2. 在浏览器中打开 `index.html`
3. 点击"设置" → "数据管理" → "开始迁移"
4. 等待迁移完成

#### 方法二：手动导出导入

**在旧版本中：**
1. 打开浏览器开发者工具（F12）
2. 执行导出脚本（见《数据迁移说明.md》）
3. 下载 JSON 文件

**在新版本中：**
1. 打开 `index.html`
2. 点击"设置" → "数据管理" → "选择文件导入"
3. 选择刚才导出的 JSON 文件

#### 注意事项

- 迁移前建议先导出备份
- 迁移后验证数据完整性
- 旧数据不会被删除，可以保留作为备份
